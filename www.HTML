What is the World Wide Web?

a. Introduction to the World Wide Web (WWW): Explain how the web works, the role of web servers, what are web browsers?

The World Wide Web (WWW), also widely known as the Web, was created by British computer scientist Sir Tim Berners-Lee. He created the web in order to have an information system where he could access all types of sources. These sources are identified by Uniform Resource Locators (URLs) . They are accessible all over the internet. The resources of the web are transferred by the HyperText Transfer Protocol (HTTP). The Web can be accessed through web browsers and are published by web servers. He was also responsible for writing HTML, URLs and HTTP.
A web browser application takes you anywhere on the internet. It retrieves information from other parts of the web and displays it on your desktop or mobile device. This information needs to be shared and displayed in a consistent format so that people using any browser, anywhere in the world can see the information. However, not all web browsers format the information in the same way.
There are two main forms of web servers. The Hardware side and the Software side. On the hardware side, the web server stores the serverâ€™s software and a website's component files (HTML, CSS, Javascript etc.) On the other hand, the software side includes several parts that control how web users access hosted files. An HTTP server is software that understands URLs (web addresses) and HTTP (the protocol your browser uses to view webpages).

Internet and Intranet

b. How is the world wide web different to the internet?

The World Wide Web is more of a system for accessing and navigating the internet. The WWW was created for information to be exchanged on a global level, whereas the internet is just a connection between multiple servers, programs and devices. If we were to visualise this relationship, In a sense, the Internet is like a global computer, and the Web is basically the user accounts on that same computer.

c. What is an intranet? How does it relate to the internet?

An intranet is a private network setup for communication purposes within an organisation. At its core, the intranet is there to help employees communicate, share information and collaborate on projects for that organisation. In the modern day world, an intranet can take on many different forms, including forums, discussion areas, and question banks, and community based groups within social media. The internet consists of all these things, but is on a global level. The intranet is more of a restricted network.

HTTP, FTP, TCP/IP

d. What is an internet protocol? compare HTTP, TCP/IP and FTP

HTTP is essentially the foundation of any data exchanging throughout the World Wide Web. The IP (Internet Protocol) is the address system of the Internet.Basic network connections are made through IP, however IP does not handle error checking. This is done through TCP (Transmission Control Protocol). TCP is the component that collects and reassembles the packets of data, while IP is responsible for making sure the packets are sent to the right destination.
FTP (File Transfer Protocol) is a way of transferring files between computers. You can use FTP to access a remote computer on the network to get or receive files, or to put or send files.
All three of these are made for data transfer within two computers that are connected to the internet, but the main differences are that while TCP contains information about what data has or has not been received yet, HTTP contains specific instructions on how to read and process this data once it arrives. On the other hand, HTTP is used to share information through the internet without any authentication. With FTP, you are required to log in to access any files on the internet/computer. FTP is a client-server protocol which looks at communication between computer to computer whereas TCP looks at the communication between computer and network.
